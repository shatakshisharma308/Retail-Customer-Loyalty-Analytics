# Retail-Customer-Loyalty-Analytics
Power BI dashboard analysing customer behaviour, revenue trends, and loyalty insights
📊 Retail Customer Behaviour & Loyalty Analytics Dashboard

This project analyses 99,000+ retail transactions to uncover customer behaviour, purchasing patterns, and loyalty insights.

🔧 Tools Used
Power BI
Excel
📌 Key Features
Interactive dashboard with KPIs: Total Revenue, Customers, Average Order Value
Customer segmentation by age, gender, and shopping behaviour
Loyalty analysis (Member vs Non-Member simulation)
Revenue trends across product categories and locations
📈 Key Insights
Certain product categories drive the majority of revenue
Loyalty members show strong contribution to overall sales
Customer spending varies significantly across age groups
Payment method trends highlight digital preference
